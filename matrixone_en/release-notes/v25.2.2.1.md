# MatrixOne v25.2.2.1 Release Report

Release date: July 13, 2025

MatrixOne version: v25.2.2.1

MatrixOne 2.2.1 ships a round of bug fixes that improve system performance, stability, and availability.

## Bug Fixes

- Optimized memory issues
- Fixed Shuffle Join in recursive CTEs
- Fixed error-checking issue in the RPC handler
- Fixed error when creating an index with a keyword
- Fixed lock-hung issue

For more details, see [MatrixOne-v25.2.2.1](https://github.com/matrixorigin/matrixone/releases/tag/v2.2.1).
