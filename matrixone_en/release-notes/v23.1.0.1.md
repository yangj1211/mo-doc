# MatrixOne v23.1.0.1 Release Report

Release date: December 16, 2023

MatrixOne version: v1.0.1

Compared to v23.1.0.0, v23.1.0.1 introduces only a few new system functions; everything else is optimization and bug fixes.

### Improvements

- Refactored code that collects tenant storage usage: [#12468](https://github.com/matrixorigin/matrixone/pull/12468)
- Improved lock-service P99 performance: [#12760](https://github.com/matrixorigin/matrixone/pull/12760)
- Improved system observability: [#12756](https://github.com/matrixorigin/matrixone/pull/12756) [#12767](https://github.com/matrixorigin/matrixone/pull/12767) [#12801](https://github.com/matrixorigin/matrixone/pull/12756) [#13282](https://github.com/matrixorigin/matrixone/pull/12756) [#13388](https://github.com/matrixorigin/matrixone/pull/12756)
- Added several debug commands: [#12772](https://github.com/matrixorigin/matrixone/pull/12772) [#12804](https://github.com/matrixorigin/matrixone/pull/12804) [#12901](https://github.com/matrixorigin/matrixone/pull/12901)
- Tuned CU calculation statistics: [#12925](https://github.com/matrixorigin/matrixone/pull/12925)
