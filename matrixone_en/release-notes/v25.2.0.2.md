# MatrixOne v25.2.0.2 Release Report

Release date: January 25, 2025

MatrixOne version: v25.2.0.2

MatrixOne 2.0.2 ships improvements and bug fixes that raise system performance, stability, and availability. The main updates are listed below.

## Improvements

- Point-query performance tuning
- Replay optimizations that reduce TN restart time
- Stats support for subscribed tables
- ETLMerge optimizations
- Improved logtail reconnection — minimize impact on CN operations and reduce CN downtime
- Implemented the `json_set` function

## Bug Fixes

- Fixed full-text-index issues
- Fixed CN hang during upgrade
- Fixed several operator-related issues
- Fixed issues under concurrent DDL
- Fixed window-function issues

For more details, see [MatrixOne-v25.2.0.2](https://github.com/matrixorigin/matrixone/releases/tag/v2.0.2).
