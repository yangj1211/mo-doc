# MatrixOne v22.0.2.0 Release Report

Congratulations! MatrixOne v22.0.2.0 was officially released on January 6, 2022. A summary of the updates follows.

## Docker

You can pull MatrixOne v0.2.0 via docker.

## Latest features

* Automatic rebalancing in the AOE columnar engine.
* Brand-new SQL parser.
* Factorized SQL execution — the fastest MPP written in Go.
* `CREATE / DROP INDEX` statements.
* `PRIMARY KEY` on table creation.
* More data types for SQL binary operators.
* `GROUP BY` / aggregate queries with equi-joins.
* New `DATE` and `DATETIME` data types.

## Known issues

* When creating many tables sequentially in a script, the cluster's `pre-allocate-group-num` parameter should be set to a larger value; otherwise you'll hit "no available raft group".
