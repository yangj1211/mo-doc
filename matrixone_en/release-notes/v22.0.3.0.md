# MatrixOne v22.0.3.0 Release Report

Congratulations! MatrixOne v22.0.3.0 was officially released on March 10, 2022. A summary of the updates follows.

## Docker

You can pull MatrixOne v0.3.0 via docker.

## Latest features

- Export data to CSV files.
- Parallel execution — faster queries.
- `IN` operator — specify multiple values in `WHERE`.
- `NULLABLE` columns in `GROUP BY`.
- `round()` and `floor()` functions.
- Chaos-testing framework in the distributed system.

## Known issues

- When creating many tables sequentially in a script, the cluster's `pre-allocate-group-num` parameter should be set to a larger value; otherwise you'll hit "no available raft group".
