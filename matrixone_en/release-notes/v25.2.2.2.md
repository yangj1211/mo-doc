# MatrixOne v25.2.2.2 Release Report

Release date: July 17, 2025

MatrixOne version: v25.2.2.2

MatrixOne 2.2.2 introduces several optimizations and bug fixes that improve system performance, stability, and usability. The main updates are listed below.

## Optimizations

- Support for adjustable CDC (Change Data Capture) frequency

## Bug Fixes

- Optimized memory issues
- Fixed JSON arbitrary value handling
- Fixed GC (garbage collector) memory pool leak
- Fixed forced GCKP (GC checkpoint) issue

For more bug-fix details, see [MatrixOne-v25.2.2.2](https://github.com/matrixorigin/matrixone/releases/tag/v2.2.2).
