# MatrixOne v24.2.0.1 Release Report

Release date: December 10, 2024

MatrixOne version: v24.2.0.1

MatrixOne 2.0.1 ships improvements and bug fixes that raise system performance, stability, and availability. The main updates are listed below.

## Improvements

- Index and query optimization
- Memory and resource optimization
- Password-management enhancements
- Backup and restore enhancements
- Experimental support for full-text search

## Bug Fixes

- Transaction management
    - Fixed deadlocks and inconsistent transaction state in distributed environments
    - Resolved race conditions in transaction processing and session management
    - Improved error handling for lock conflicts and concurrent delete operations

- SQL execution
    - Fixed issues with recursive CTEs, view creation, and `ON DUPLICATE KEY UPDATE`
    - Resolved incorrect row counts during `LOAD DATA` and compressed-file handling

- Logtail and cache
    - Resolved data-race conditions and performance issues in logtail processing
    - Fixed cache-usage and cache-hit-status monitoring

- Full-text index
    - Fixed panics related to full-text-index creation and runtime filters
    - Improved drop operations on tables carrying full-text indexes

- Backup and restore
    - Fixed missing workspace data during restore
    - Fixed errors when restoring tables or databases with special keywords or case-sensitive names

- Performance and stability
    - Fixed memory-allocation issues in batch and dedup operations
    - Resolved panics and race conditions across query execution and system diagnostics
    - Plus many smaller bug fixes that further improve reliability — see the full changelog for details

For more details, see [MatrixOne-v24.2.0.1](https://github.com/matrixorigin/matrixone/releases/tag/v2.0.1).
