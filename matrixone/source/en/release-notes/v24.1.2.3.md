# MatrixOne v24.1.2.3 Release Report

Release date: September 11, 2024

MatrixOne version: v24.1.2.3

Compared to v24.1.2.2, v24.1.2.3 focuses on bug fixes, performance tuning, and minor feature polish — no major new features.

## Improvements

- Memory optimization: better memory usage for dedup checks, SEMI Join, and TableScan operations.
- Enhanced logging: added logs for account restrictions, account pausing, and other operations — improving debuggability and observability.
- Performance: optimized handling of `LIMIT 0` queries, improved dynamic cache management, and raised query performance for JSON sorting.

## Bug Fixes

v24.1.2.3 addresses multiple issues related to memory usage, transaction handling, and proxy-connection stability — raising MatrixOne's overall stability and reliability. See [MatrixOne-v24.1.2.3](https://github.com/matrixorigin/matrixone/releases/tag/v1.2.3) for the full list.
