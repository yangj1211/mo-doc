# MatrixOne v24.1.2.2 Release Report

Release date: July 12, 2024

MatrixOne version: v24.1.2.2

Compared to v24.1.2.1, v24.1.2.2 introduces no new features — it focuses on bug fixes and optimizations.

## Improvements

- Support `GROUP BY 1`, `MAKEDATE`.
- Added runtime metrics.
- Support reloading the auto-increment cache.
- Async loading of file info for the disk cache.
- Tuned flush and merge.

## Bug Fixes

Fixes around partition state, proxy, MySQL compatibility, lock service, stats, and merge memory control. See [MatrixOne-v24.1.2.2](https://github.com/matrixorigin/matrixone/releases/tag/v1.2.2) for details.
