# MatrixOne v23.1.0.2 Release Report

Release date: December 23, 2023

MatrixOne version: v23.1.0.2

Compared to v23.1.0.1, v23.1.0.2 introduces only a few new system functions and improvements — it primarily focuses on bug fixes.

### Improvements

- Support for equality transitivity: [#13596](https://github.com/matrixorigin/matrixone/pull/13596)

### Bug Fixes

- Fixed reserved-keyword errors: [#13561](https://github.com/matrixorigin/matrixone/pull/13561)
- Fixed prefetch-delete error: [#13590](https://github.com/matrixorigin/matrixone/pull/13590)
- Fixed blob-related errors: [#13593](https://github.com/matrixorigin/matrixone/pull/13593)
- Fixed mo_ctl merge hang: [#13601](https://github.com/matrixorigin/matrixone/pull/13601)
- Fixed incorrect pushdown of runtime filters on left join: [#13661](https://github.com/matrixorigin/matrixone/pull/13661)
- Fixed package partitioning not considering constant vectors: [#13607](https://github.com/matrixorigin/matrixone/pull/13607)
- Fixed invalid-symlink error in local FileService list: [#13675](https://github.com/matrixorigin/matrixone/pull/13675)
- Fixed abnormal interruption during cache eviction: [#13696](https://github.com/matrixorigin/matrixone/pull/13696) [#13706](https://github.com/matrixorigin/matrixone/pull/13706) [#13708](https://github.com/matrixorigin/matrixone/pull/13708)
- Fixed `SHOW SUBSCRIPTIONS` failure in some scenarios: [#13656](https://github.com/matrixorigin/matrixone/pull/13656) [#13599](https://github.com/matrixorigin/matrixone/pull/13656)
- Fixed wrong deletion of dirty-data blocks: [#13647](https://github.com/matrixorigin/matrixone/pull/13647)

### Features

- Added `LCASE`, `UCASE` system functions: [#13686](https://github.com/matrixorigin/matrixone/pull/13686)
