# MatrixOne v25.2.2.0 Release Report

Release date: June 18, 2025

MatrixOne version: v25.2.2.0

MatrixOne 2.2.0 ships a round of bug fixes that further improve system performance, stability, and availability.

## Bug Fixes

- Fixed CDC-related bugs
- Fixed enum-type bug
- Fixed duplicate primary-key issue caused by large-scale deletes
- Fixed Rollup issue with the HAVING clause
- Fixed join-order issue in DML plans

For more details, see [MatrixOne-v25.2.2.0](https://github.com/matrixorigin/matrixone/releases/tag/v2.2.0).
