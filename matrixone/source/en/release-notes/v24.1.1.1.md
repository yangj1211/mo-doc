# MatrixOne v24.1.1.1 Release Report

Release date: February 4, 2024

MatrixOne version: v24.1.1.1

Compared to v23.1.1.0, v24.1.1.1 introduces no new features — it focuses on improvements and bug fixes.

### Improvements

- Stability gains from fixes related to the lock service and proxy.
- Enhanced forward compatibility.
- MySQL compatibility polish.
- More accurate runtime resource-consumption statistics.
