# MatrixOne v22.0.5.1 Release Report

Congratulations! MatrixOne v22.0.5.1 was officially released on August 19, 2022. This version resolves several log-replay and storage garbage-collection (GC) issues.

## Improvements

- Optimized checkpoint replay. #4214
- Fixed a defect where block and index file reference counts weren't released. #4052
- Fixed a replay dead-loop defect when logs were not full. #4051

With these fixes, MatrixOne instance stability is improved.
