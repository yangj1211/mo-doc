# MatrixOne v22.0.4.0 Release Report

Congratulations! MatrixOne v22.0.4.0 was officially released on May 5, 2022. A brief summary of the updates follows.

## Docker

You can pull MatrixOne v0.4.0 via docker:

```
docker pull matrixorigin/matrixone:0.4.0
```

## Latest features

- Inner joins with equality operators.
- `FROM` subqueries.
- `decimal` type.
- New system functions:
    - Math: Abs, Log, Ln, Ceil, Exp, Power, Pi, Sin, Sinh, Cos, ACos, Tan, ATan, Cot
    - Date/Time: Month, Weekday, Date, DayOfYear
    - String: Space, Reverse, Substring, Ltrim, Rtrim, StartsWith, EndsWith, Lpad, Rpad, Empty, LengthUTF8
- New aggregates: Bit_and, Bit_or, Bit_xor, Stddev_pop, Var.

## Known issues

- When creating many tables sequentially in a script, the cluster's `pre-allocate-group-num` parameter should be set to a larger value; otherwise you'll hit "no available raft group".
