# MatrixOne v25.2.1.1 Release Report

Release date: May 8, 2025

MatrixOne version: v25.2.1.1

MatrixOne 2.1.1 ships a mix of improvements and bug fixes that raise system performance, stability, and usability. The main updates are listed below.

## Improvements

- Optimized join ordering and enabled runtime filtering for full-text updates
- Added local transaction-timeout checks
- Automatic retry on remote-lock I/O timeouts
- Converted BETWEEN filters on composite keys to PREFIX_BETWEEN

## Bug Fixes

- Fixed OOM-related errors
- Fixed issues in partial merge operations
- Fixed bugs in CDC (Change Data Capture)
- Fixed bug related to single-column CLUSTER BY keys
- Fixed issue in large-scale delete operations
- Fixed crash when sorting columns of type UUID
- Fixed over-handling of deadlocks

For more details, see [MatrixOne-v25.2.1.1](https://github.com/matrixorigin/matrixone/releases/tag/v2.1.1).
