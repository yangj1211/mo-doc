# MatrixOne v25.2.0.3 Release Report

Release date: March 12, 2025

MatrixOne version: v25.2.0.3

MatrixOne v25.2.0.3 ships improvements and bug fixes that raise system performance, stability, and availability. The main updates are listed below.

## Improvements

- Support for new time hints
- Minor improvements to loop join
- Implemented `information_schema.table_constraints`
- Added flush on send-prepared-response
- Added transaction-state checks

## Bug Fixes

- Fixed OOM-related errors
- Fixed CN hang during leak-check
- Fixed a few zm merge bugs
- Fixed bug related to concurrent DROP TABLE

For more details, see [MatrixOne-v25.2.0.3](https://github.com/matrixorigin/matrixone/releases/tag/v2.0.3).
