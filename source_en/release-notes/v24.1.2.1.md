# MatrixOne v24.1.2.1 Release Report

Release date: June 30, 2024

MatrixOne version: v24.1.2.1

Compared to v24.1.2.0, v24.1.2.1 introduces no new features — it focuses on bug fixes and optimizations.

## Improvements

- In-memory cache now supports multiple memory allocators.
- Optimized `show accounts` performance.
- Tuned `fileservice` observability metrics.
- Incremental backup support.
- Tuned TCP packet size estimation.
- Refactored `mologing`.
- Support restoring system tables.
- Support zonemap-based manual merge.

## Bug Fixes

Fixes around snapshot reads, MySQL compatibility, lock service, runtime filters, and system-table upgrade. See [MatrixOne-v24.1.2.1](https://github.com/matrixorigin/matrixone/releases/tag/v1.2.1) for details.
