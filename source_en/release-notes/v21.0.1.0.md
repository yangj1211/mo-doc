# MatrixOne v21.0.1.0 Release Notes

Congratulations! MatrixOne v21.0.1.0 was officially released on October 24, 2021.

## Docker

You can pull MatrixOne v0.1.0 via docker.

## Core features

### SQL

This release supports the following SQL statements:

### DDL

* CREATE / DROP DATABASE
* CREATE / DROP TABLE

### DML

* INSERT
* LOAD DATA
* SELECT
    - WHERE
    - GROUP BY
    - ORDER BY
    - LIMIT, OFFSET
* SHOW
    - DATABASES
    - TABLES
* USE

### Data types

* TINYINT / SMALLINT / INT / BIGINT, SIGNED / UNSIGNED
* FLOAT / DOUBLE
* CHAR / VARCHAR

### Operators

* &lt;, >=, <, <=, <>, !=, =
* BETWEEN … AND …, NOT BETWEEN … AND …
* AND, &&, OR, ||
* +, -, *, /, %, MOD, DIV, NEG
* CAST

### Aggregates

* COUNT
* SUM
* AVG
* MAX
* MIN

## Data engine

* MySQL syntax support.
* Built-in AOE columnar engine — a unified database engine.
* Real-time analytical queries.
* MPP-based vectorized query engine.
* Partial vectorized execution via SIMD instructions.
* Strongly-consistent distributed AOE engine built on the Raft protocol.
* Significant replication-state-machine performance gains from a reuse design between the Raft log and the Write-Ahead Log.
