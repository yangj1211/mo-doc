## Common acronyms

- **CN** (Compute Node) — stateless compute node
- **TN** (Transaction Node) — transaction node
- **HAKeeper** — high-availability scheduling component
- **LogService** — Raft-based log service
- **TAE** (Transactional Analytical Engine) — MatrixOne's in-house storage engine
- **PITR** (Point-in-Time Recovery)
- **CDC** (Change Data Capture)
- **HTAP** (Hybrid Transaction / Analytical Processing)
- **HSTAP** — Hybrid Streaming / Transaction / Analytical Processing (MatrixOne extension)
- **Logtail** — uncompacted recent transaction logs held by LogService
- **Stage** — external-storage path object
- **Datalink** — MatrixOne's reference type for storage files
